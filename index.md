# Operating Systems Coursework Journal

## Student Information
- Module: Operating Systems
- Course: Computer Science
- Platform: Ubuntu Server 24.04 (VirtualBox)

## Project Overview
This coursework documents the deployment, configuration, security hardening, and performance evaluation of a headless Linux server operating system administered remotely via SSH.

The project follows a seven-phase structure covering system planning, security implementation, monitoring, performance evaluation, and security auditing.
